# 📈 Optimization Stock - Stock Market Analysis Dashboard

## 📌 Project Overview

OptiStock is an interactive Stock Market Analysis Dashboard built using **Power BI** to analyze historical stock market data and provide meaningful business insights for investors and financial analysts.

The dashboard helps users monitor stock performance, identify market trends, compare company performance, and make data-driven investment decisions through dynamic visualizations.

---

## 🎯 Objectives

- Analyze historical stock market data.
- Compare stock performance across companies.
- Track opening, closing, highest, and lowest prices.
- Identify profitable investment opportunities.
- Visualize trends using interactive dashboards.
- Enable data-driven financial decision making.

---

## 🛠️ Tech Stack

- **Power BI**
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset
- Data Visualization
- Data Cleaning & Transformation

## 📊 Dashboard Features

- 📈 Stock Price Trend Analysis
- 💹 Opening vs Closing Price Comparison

- 📉 Highest and Lowest Price Analysis

- 🎯 Interactive Filters & Slicers
